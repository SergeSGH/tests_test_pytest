1. Проверка работы метода GET:
   1) GET запрос на URL `/new_post/`, далее:
      а) проверка статуса ответа
      б) проверка наличия аттрибута 'form' в аттрибуте 'context' объекта response
      в) проверка наличия всех необходимых полей объекта 'form' и корректности их типа
         (CharField, TextField, ImageField, required...)

2. Проверка работы метода POST:
   1) POST запрос на URL `/new_post/`, с корректными данными, далее:
      а) проверка статуса ответа - 301 или 302 (перенаправление на нужную страницу в случае валидности формы)
      б) проверка url страницы, на которую перенаправлен запрос
      в) проверка появления нового объекта (поста)
   2) POST запрос на URL `/new_post/`, с некорректными данными, далее:
      а) проверка статуса ответа - 200 (в случае невалидности формы возвращается HTML страница с полями формы)
      б) проверка, что новый объект (пост) не создается
